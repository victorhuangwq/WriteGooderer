WriteGooderer

Demo & Test Page — Every text field scenario in one place. Load the extension, open this page, and test away.

How to Test

  1. Build the extension: npm run build
  2. Load dist/ as an unpacked extension in chrome://extensions
  3. Open this page (demo.html) in Chrome
  4. Click into any text field — the "W" icon should appear on eligible fields
  5. Click "W" to open the popup, then try Proofread or Change Tone
  6. Fields marked NO ICON should NOT show the "W" icon

Edge Cases

Empty Textarea

Clicking proofread on empty text should be a no-op (the button shouldn't submit an empty request).

Perfect Text

Should score 85+ and show "no changes needed" or minimal corrections.

Very Long Text

Tests model token limits. May produce partial results or an error if text is too long.

Mixed Languages

Tests how the AI handles non-English text mixed with English.

All Caps / Unconventional Casing

Should handle text that's ALL CAPS or has weird casing patterns.
